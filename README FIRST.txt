[ REQUIREMENTS ]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• Windows 10 / 11  
• Internet connection  
• **Latest version of Node.js (MANDATORY!)**

  ⚠️ WARNING:  
  If you have an older Node.js version installed,  
  **UNINSTALL IT FIRST** before installing the latest one.

  The program will NOT work correctly with outdated versions.


[ INSTALLING NODE.JS ]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1) Go to: https://nodejs.org  
2) Download the **latest LTS version**  
3) Install normally  
4) Restart your computer (recommended)


[ STARTING THE PROGRAM FOR THE FIRST CHECK] 
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1) Open **Command Prompt as Administrator**  
   (Right-click the icon → “Run as administrator”)

2) Navigate to the project folder:

    cd C:\Path\To\Your\Project

3) Start the program:

    npm run dev

Or, for beginners:

    Double-click **Start.bat**



[ CONFIGURE THE .ENV FILE ]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Open the following file inside the project folder:

    .env

Fill in **ALL required fields**:

• BOT_TOKEN           → Your Telegram bot token  
• MINI_APP_URL        → Mini-app URL  
• COMMUNITY_URL       → Your community/community channel link  
• CHAT_ID             → Telegram chat ID  

⚠️ Missing values may cause startup errors.




[ AUTOMATIC UPDATES ]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
– Do NOT manually update dependencies  
– Do NOT delete or modify `node_modules`  
– Do NOT run `npm update`

✔ The project includes a built-in **Auto-Update System**  
  Every launch automatically checks for updates and applies them when needed.  
  No manual actions required.
